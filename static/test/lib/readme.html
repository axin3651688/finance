Welcome to Wijmo Enterprise, a new generation of JavaScript controls.

This zip file contains three folders:

  • Dist contains Wijmo Core distributable files, including the Wijmo controls, CSS styles, and directives for use in AngularJS applications.
  • Samples contains sample applications that show the Wijmo controls in action.
  • VsTemplates contains Visual Studio project templates for Wijmo applications.

To get started, check out the samples and see how to use Wijmo in your applications. Happy coding!

If you need additional help, check out the Wijmo Documentation, or to check the latest updates, visit the Wijmo Core page.
